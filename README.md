
# Visionária.dev 🌐✨

Seja bem-vindo ao **Visionária.dev** — um blog pessoal criado para compartilhar ideias, aprendizados e experiências no mundo da tecnologia, com foco em desenvolvimento web, inovação e crescimento contínuo.

## 🖥 Sobre o Projeto

Este é um projeto pessoal desenvolvido com HTML5 e CSS3, pensado para ser totalmente responsivo e acessível em diferentes tamanhos de tela (mobile, tablet e desktop).

O objetivo principal do blog é ser um espaço de expressão, onde conceitos técnicos são abordados de forma clara, didática e com propósito — especialmente nas áreas de front-end, back-end e carreira na área de tecnologia.

## 📁 Estrutura do Projeto

```
visionaria.dev/
├── assets/
│   ├── img/
│   │   └── perfil linkedin.png
│   └── style.css
├── index.html
└── README.md
```

## ✨ Funcionalidades

- Página inicial com saudação e propósito do blog.
- Seção "Sobre mim" com informações da autora.
- Área de artigos com textos informativos.
- Menu de navegação com rolagem suave por âncoras.
- Links úteis com recursos para aprofundar conhecimentos.

## 📱 Responsividade

A interface foi projetada para se adaptar a diversos dispositivos e tamanhos de tela usando media queries no CSS, garantindo uma boa experiência tanto em dispositivos móveis quanto em desktops.

## 🧪 Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts (Open Sans)
- Responsividade com media queries

## 🚀 Como visualizar o projeto

1. Clone ou baixe o repositório:
```bash
git clone https://github.com/seu-usuario/visionaria.dev.git
```

2. Navegue até o diretório:
```bash
cd visionaria.dev
```

3. Abra o arquivo `index.html` em qualquer navegador moderno.
